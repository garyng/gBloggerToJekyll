 Javascript代码：javascript: document.body.contentEditable = 'true';
document.designMode = 'on'; void 0  
  
  
 使用方法：  
 1、直接把代码贴在URL栏上！  
 注：有些浏览器会把代码前的"Javascript:"去除，必需手动添加！  
  
  
 2、大大力地按下键盘上的Enter键吧！！  
  
 3、可以随意编辑咯～
